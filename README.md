# oscm-commons



You can use it as follows:
```
<repositories>
  <repository>
  <id>jitpack.io</id>
  <url>https://jitpack.io</url>
  </repository>
</repositories>

<dependency>
  <groupId>com.github.servicecatalog</groupId>
  <artifactId>oscm-commons</artifactId>
  <version>1.3</version>
</dependency>
```
or more specific
```
<repositories>
  <repository>
  <id>jitpack.io</id>
  <url>https://jitpack.io</url>
  </repository>
</repositories>

<dependency>
  <groupId>com.github.servicecatalog.oscm-commons</groupId>
  <artifactId>oscm-common</artifactId>
  <version>1.3</version>
</dependency>
```
